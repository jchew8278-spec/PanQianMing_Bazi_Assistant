# 潘芊名命理解说助手 🌿

一款为命理师定制的轻量 Python 助手，可在 Replit 或 GitHub Codespaces 上运行。

### ✨ 功能特色
- 自动计算八字五行比例（支持农历输入或自动转换）
- 单年流年、区间流年分析（2026–2036）
- 水晶与精油能量建议
- 数字命理分析（门牌、手机、车牌、英文名）
- 自动推荐 4 位有利号码
- 输出风格温柔自然，附专属署名

---

### 🚀 使用方法（在 Replit 上）
1. 打开 [https://replit.com](https://replit.com) 登录（建议用 GitHub 登录）  
2. 新建一个 Python 项目  
3. 上传或粘贴以下文件：  
   - main.py  
   - requirements.txt  
   - data/crystals.json  
   - data/oils.json  
4. 点击 **Run** ▶️ 运行  
5. 按提示输入命主信息、选择功能即可生成完整解说。
